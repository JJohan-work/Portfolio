Tricorder

About

This is a project I have been working on since the beginning of 2020. It is a small pocket PC with a clamshell design. It is made mostly of off the shelf parts from Adafruit. It has two screens, USB cababilites, a software defined radio, GPS and upwards of 5 hours of battery. Its main computer is a Raspberry Pi Zero currently running mostly unchanged Raspberry Pi OS and scripts written in python code for its peripherials. The external shell was modelled by me and is based off the shape of the Tricorder from Star Trek

Purpose

This is a continuation of past work in 3d design, 3d manufacturing and complex electronic devices. It is meant to be both a functional tool and a display of my current abilites

Goals

Currently I am looking to expand my coding skills to program both a user interface and more advanced uses from its multiple sensors and outputs.
